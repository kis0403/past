Machine Learning을 위한 Tensorflow 환경 설치 (windows,GPU)

매일 4GB RAM으로 고통받다가 드디어 데스크탑을 새로 구매하였습니다.

tensorflow의 GPU환경도 사용해보기 위해 그래픽 카드도 맞췄습니다. 그 기념으로(?) Tensorflow GPU환경을 설치하는 법에 대해 포스팅하려고 합니다.

설치환경

OS : Windows 10 Home 64bit(10.0 빌드 10586)

그래픽카드 : NVIDIA GeForce GTX 1050

프로세서 : AMD Ryzen 5 1600 Six-core Processor

 

 

1. python설치

최신 버전의 경우 호환성 문제가 발생할 수 있습니다. 저같은 경우는 3.6.1버젼으로 받았습니다.

https://www.python.org/downloads/

2. Anaconda설치

4.4.0버전으로 받았습니다.

https://www.continuum.io/downloads

3. tensorflow 환경 구축

Anaconda Prompt창에서

1
conda create -n tensorflow python=3.6

를 입력해서 tensorflow라는 이름의 새로운  파이썬 환경을 만들고 y를 입력해주시면 설치를 시작하게 됩니다.

4. tensorflow GPU환경 구축을 위한 CUDA Toolkit, cuDNN v5.1 두 가지 프로그램을 설치해야 합니다.

CUDA Toolkit https://developer.nvidia.com/cuda-downloads

cuDNN https://developer.nvidia.com/rdp/form/cudnn-download-survey

cuDNN의 경우 Nvidia Developer사이트의 회원 가입이 필요하므로 가입하시면 됩니다.

주의하실 점은 윈도우 10환경에서는 Download cuDNN v5.1 (Jan 20, 2017), for CUDA 8.0 즉, 5.1버전으로 다운로드 받으셔야 오류가 발생하지 않습니다. (몰라서 한참 찾아봤습니다 ㅠ)

이제 다시 anaconda prompt로 돌아오셔서

activate tensorflow

를 입력하셔서 방금 전 만들었던 환경으로 접속합니다.

5. tensorflow 설치

입력 라인 앞 부분에 (tensorflow)로 바뀐것을 볼 수 있습니다.

1
pip install --ignore-installed --upgrade https://storage.googleapis.com/tensorflow/windows/gpu/tensorflow_gpu-1.2.1-cp36-cp36m-win_amd64.whl

윗 라인을 입력하면 1.2.1버전의 tensorflow가 설치되는 것을 볼 수 있습니다.

이제 python을 적어 파이썬을 실행한 다음

1
2
3
4
import tensorflow as tf
hello = tf.constant('Hello, TensorFlow!')
sess = tf.Session()
print(sess.run(hello))

를 입력하시면 Hello, TensorFlow!가 출력됩니다.

 

 

만약 오류가 발생한다면 오류 메시지를 구글링하여 원인을 찾아야 합니다. 그렇게 찾던 도중 자가 진단 코드를 찾게되었는데 유용하게 쓰여서 여기도 링크를 남기겠습니다.

https://gist.github.com/mrry/ee5dbcfdd045fa48a27d56664411d41c

이전에 설치한 tensorflow환경에서 py 파일을 실행하시면 결과가 나타납니다.
