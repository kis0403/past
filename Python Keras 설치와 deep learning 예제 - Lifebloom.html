Python Keras 설치와 deep learning 예제

keras는 Neural Network에서 tensorflow나 Theano와 같은 좋은 성능을 내는 python 기반 api입니다.

이번 시간에는 간단한 keras의 내장 예제를 통해서 어떤 식으로 구동되는지 알아보겠습니다.

 

1. Prediction

먼저 아나콘다 프롬프트에서 keras를 설치한뒤 jupyter notebook을 실행합니다.

2
import keras
import numpy as np

from keras.datasets import boston_housing, mnist, cifar10, imdb

여기서 X_train은 13개의 변수와 404개의 행, Y_train은 404개의 1변수 데이터를 의미합니다.

y_train[0] # numerical target => regression problem

와 같은 수치형의 자료이므로  회귀 예측 문제로 분류합니다.

필요한 패키지를 로드하고

 

1
2
3
4
model = Sequential()
model.add(Dense(13, input_dim = x_train.shape[1], kernel_initializer = 'normal', activation = 'relu'))
model.add(Dense(7, kernel_initializer = 'normal', activation = 'relu'))
model.add(Dense(1, kernel_initializer = 'normal'))

다음과 같이 모델을 생성합니다 이때, Sequantial은 인공신경망 모델을 생성하기 위해 쓰이는 linear layer stack입니다.

그 다음 Dense는 fully connected layers를 생성하는데 쓰이는데 위에서는 노드의 크기가 13,7,1인 fcl를 생성합니다.

kernel_initializer는 초기 랜덤 가중치를 어떻게 설정할 것인지 지정하는 옵션으로 다양한 방식이 있습니다. – https://keras.io/initializers/

활성화 함수(activation)은 ReLU(Rectified Linear Unit)로 지정합니다. –  https://keras.io/activations/

 

1
model.compile(loss = 'mean_squared_error', optimizer = 'adam')

인공신경망을 생성한 다음 모델을 컴파일링 하기 위해 다음과 같이 입력합니다.

loss function은 mse를 사용하였고, optimizer는 ADAM으로 하였습니다. – https://keras.io/optimizers/      ,  https://keras.io/metrics/

Metric 옵션을 추가할 수도 있는데 Metric을 계산할 때 Metric을 평가 한 결과가 사용되지 않는다는 점을 제외하면 Metric function은 loss function과 유사합니다.

 

1
model.fit(x_train, y_train, batch_size = 30, epochs = 15, verbose = 1)

이제 본격적으로 학습을 시작합니다.

batch-size는 gradient update당 사용되는 샘플의 수 이고, epoch은 training data array를 반복하는 횟수입니다.

verbose는 진행과정을 표시하는 옵션으로 0,1,2 중에 선택하시면 됩니다.

y_pred = model.predict(x_test)
mean_squared_error(y_pred, y_test) # regression error

model = Sequential()

model.add(Convolution2D(32, 3, 3, activation='relu'))
model.add(MaxPooling2D(pool_size=(2,2)))
model.add(Dropout(0.25))
model.add(Flatten())
model.add(Dense(128, activation='relu'))
model.add(Dropout(0.5))
model.add(Dense(10, activation='softmax'))

이제 모델을 만드는 과정입니다. pooling, dropout, conv2D 에 대해 설명하게되면 너무 길어지고 깊게 들어가기 때문에 다른 포스팅에서 다루려고 합니다.

여기서는 단순히 여러개의 층을 만들어 train data를 통과 시키는 걸로 알고 넘어가겠습니다.

예제를 수행할 때 참고할만한 점은 keras가 tensorflow기반인 경우와 theano 기반인 경우 dimension order가 다르다는 점입니다.

위의 X_train 데이터는 tensorflow에서 사용하는 형식인 N(umber of images),H(eight),W(idth),C(hannel) 즉 NHWC 순서로 되어있는데 반해 theano는 NCHW 순서를 사용하기 때문에 차이가 발생합니다. 인터넷에서 코드를 참고하실 때 주의하시기 바랍니다.
