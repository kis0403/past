Regression with Machine Learning 4. Regularization for spartsity(희소 학습)

제곱을 활용하는 l2 norm 과 다르게 절대값을 활용하는 l1 norm에 대해 알아보겠습니다.

minθJLS(θ)subject to∥θ∥1≤R

 \sum_{\overset{b}}

벡터 θ=(θ1,…,θb)⊤ 의 l1 norm, 즉 ∥θ∥1은 벡터 θ의 각 요소의 절대값의 합으로 정의된다.

∥θ∥1=∑bj=1∣θj∣

 

l1에 대한 증명은 l2에 비해 복잡하므로 생략하고(구글링…) 간단한 특징을 나열하면 파라미터 공간의 축 상에 존재하기가 쉽기 때문에 특정 파라미터의 값이 0이 되는 경우가 많이 발생합니다.

(축 상에 존재할 가능성은 꼭지점이 있는 부분에서 접하기 때문) 그래서 feature selection 기능을 하여 사용되는 변수를 줄여주는 역할을 할 수 있습니다.

 

이와 다르게 데이터의 열이 행보다 많은 경우에는 l1의 성능이 떨어지는 경우가 많습니다. 그래서 두 norm을 더해 단점을 보완한 elastic net regularization 등과 같은 방법도 있습니다.
