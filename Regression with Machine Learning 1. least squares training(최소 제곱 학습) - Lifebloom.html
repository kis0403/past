Regression with Machine Learning 1. least squares training(최소 제곱 학습)

최소 제곱 학습은 JLS(θ)=12∑ni=1(fθ(xi)−yi)2가 최소가 되게 하는 parameter θ를 학습

fθ(xi) : model output , {yi}ni=1 : train output

 

θ^LS=argminθJLS(θ)

LS : Least Squares,  fθ(xi)−yi : residual(잔차) , 제곱 오차 (fθ(xi)−yi)2가 residual의 l2 norm 이라서 l2 손실 최소화 학습으로도 불림

다음과 같은 linear model에서,

fθ(x)=∑bj=1θjϕj(x)=θ⊤ϕ(x)

train error JLS is…

JLS(θ)=24∥Φθ−y∥2

y=(y1,…,yn)⊤ n th vector of train output , Φ : n×b design matrix

Φ=⎛⎝⎜⎜ϕ1(x1)⋮ϕ1(xn)⋯⋱⋯ϕb(x1)⋮ϕb(xn)⎞⎠⎟⎟

편미분 about parameter vector θ of JLS  : ∇θJLS

 ∇θJLS=(∂JLS∂θ1,…,∂JLS∂θb)⊤=Φ⊤Φθ−Φ⊤y

이 식을 0으로 만드는 LS의 해가 있다면 다음의 Equation을 만족한다.

Φ⊤Φθ=Φ⊤y

이 방정식의 해인 θ^LS는 design matrix Φ의 일반 역행렬인 Φ†을 이용해 다음과 같이 구할 수 있다.

θ^LS=Φ†y

Φ⊤Φ가 역행렬을 가진다면 다음과 같이 나타낼 수 있다.

Φ†=(Φ⊤Φ)−1Φ⊤

 

i번째 train sample에 대한 Squared Error에 weight wi≥0을 부여한 LS training을 weighted LS training이라 부른다.

minθ12∑ni=1wi(fθ(xi)−yi)2

weighted LS 학습의 해는 다음과 같이 구할 수 있다.

(Φ⊤WΦ)†Φ⊤Wy

이 때, W=(w1,…,wn)⊤

 

fθ(x)=∑nj=1θjK(x,xj)

design matrix Φ를 다음 식과 같이 정의되는 Kernal Matrix K와 치환하여 linear model과 같은 방법으로 해를 구할 수 있다.

K=⎛⎝⎜⎜K(x1,x1)⋮K(xn,x1)⋯⋱⋯K(x1,xn)⋮K(xn,xn)⎞⎠⎟⎟
